Hero RPG (Role-Playing Game)

February 2022 | GitHub

Terminal-based RPG

My first project during the bootcamp and my first experience coding. We were tasked with creating an RPG using Python and Object-Oriented Programming that runs within the terminal. It was a challenge getting the logic to work, but with some late-night dedication and caffeine I managed to make a project I was proud of. Especially for my first time. Since we were given creative control, I would say that I had an enjoyable time creating character names and menu options. There's still more I would like to modify and alter, but due to the quick progression on the bootcamp I haven't been able to go back to it just yet.

  • First Implementation of Python
  • First use of Object-Oriented Programming
  • Learned how to use IF and ELIF, WHILE, CLASS, and more

Blackjack

March 2022 | GitHub

Front-End use of JavaScript

This was a solo project where we used HTML and CSS elements for the first time. I had to create the logic for a game of Blackjack while using CSS styling to make it look nice. I quickly learned that the design aspects of web design comes more naturally to me. In regards to logic, Blackjack turns out to be more in-depth that I realized and I learned that sometimes the simple things we take for granted in the physical world are far more complicated when turned into code.

  • Learned to select DOM elements using JavaScript
  • Adding event listeners
  • Creating new DOM elements, setting DOM element Attributes
  • Introduction to HTML and CSS elements
